# S-02: Friend Connections

Users can invite another registered user (by email) to connect, the recipient can accept or reject the invitation, and either side of an accepted connection can remove it.

## Roadmap link

Roadmap item S-02 (`friend-connections`) — parallel with S-01 (collection-management), both gated only by F-02 (auth-scaffold). Unlocks S-03 (friend-discovery), which requires a confirmed friend connection as its access gate.
